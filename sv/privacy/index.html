// Juridiskt

Integritetspolicy

Senast uppdaterad: mars 2026

HyperCubeSphere ("vi," "vår" eller "oss") är engagerade i att skydda din integritet. Denna policy beskriver hur vi samlar in, använder och skyddar din personliga information när du besöker vår webbplats eller använder våra tjänster.

// 01

Information Vi Samlar In

Vi samlar in information som du tillhandahåller direkt, såsom ditt namn, e-postadress, företagsnamn och meddelandeinnehåll när du använder vårt kontaktformulär. Vi samlar även in standarddata för webbanalys, inklusive IP-adresser, webbläsartyp, besökta sidor och tid spenderad på vår webbplats. Vi samlar inte in känsliga personuppgifter om de inte uttryckligen tillhandahålls av dig i samband med ett konsultuppdrag.

// 02

Hur Vi Använder Din Information

Vi använder insamlad information för att svara på dina förfrågningar, tillhandahålla begärda tjänster, förbättra vår webbplats och användarupplevelse, skicka relevanta uppdateringar om våra tjänster (med ditt samtycke), efterleva lagkrav och skydda säkerheten för våra system och användare.

// 03

Datadelning och Utlämnande

Vi säljer, byter eller hyr inte ut din personliga information till tredje parter. Vi kan dela data med betrodda tjänsteleverantörer som hjälper till att driva vår webbplats och bedriva vår verksamhet (under sekretessavtal), när det krävs enligt lag eller rättsliga förfaranden, eller för att skydda rättigheterna, egendomen eller säkerheten för HyperCubeSphere, våra kunder eller andra.

// 04

Datasäkerhet

Som ett cybersäkerhetsfokuserat företag tar vi dataskydd på allvar. Vi implementerar branschstandardsäkerhetsåtgärder inklusive kryptering under överföring (TLS 1.3) och i vila, åtkomstkontroller baserade på principen om minsta behörighet, regelbundna säkerhetsgranskningar och penetrationstester samt incidenthanteringsrutiner. Dock är ingen metod för elektronisk överföring eller lagring 100% säker, och vi kan inte garantera absolut säkerhet.

// 05

Dina Rättigheter

Beroende på din jurisdiktion kan du ha rätt att få tillgång till, rätta eller radera dina personuppgifter, invända mot eller begränsa behandling av dina data, dataportabilitet, återkalla samtycke när som helst och lämna in ett klagomål till en tillsynsmyndighet. För att utöva dessa rättigheter, kontakta oss på privacy@hypercubesphere.online.

// 06

Cookies

Vi använder nödvändiga cookies för att säkerställa att vår webbplats fungerar korrekt. Vi kan använda analyticscookies för att förstå hur besökare interagerar med vår webbplats. Du kan hantera cookiepreferenser via dina webbläsarinställningar. Vi använder inte cookies för reklam- eller spårningsändamål.

// 07

Ändringar av Denna Policy

Vi kan uppdatera denna integritetspolicy från tid till annan för att återspegla ändringar i våra metoder eller tillämplig lagstiftning. Vi kommer att publicera den uppdaterade policyn på denna sida med ett reviderat "senast uppdaterad"-datum. Vi uppmuntrar dig att granska denna policy regelbundet.

// 08

Kontakta Oss

Om du har frågor om denna integritetspolicy eller våra datametoder, vänligen kontakta oss på privacy@hypercubesphere.online. För säkerhetsrelaterade frågor, kontakta security@hypercubesphere.online.
